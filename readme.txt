== Description ==

An easy way to generate and configure a widget for your HolidayCheck-ratings or -recommendations. 

Just copy & paste the URL of your hotel at HolidayCheck, select the type of widget you want to use and you are done!

= Options =

* Widget title
* Language (DE, AT, CH, EN, FR, IT, ES, PL)
* Type (Recommendation rate, Reviews with or without ratings)
* Widget size (width: 150px, 300px, 600px)
* Number of reviews (1, 3, 5)
* Color theme (Blue, grey, white)
* Show "Review now"-button (yes or no)

Plugin comes with a German language file.

== Installation ==

1. Download the ZIP-file
2. Copy it to your WordPress-plugin-folder on your server
3. Activate the plugin
4. Go to Appearance --> Widgets
5. Select the widget "Generator for HolidayCheck Widgets"
6. Enter the URL of your hotel at HolidayCheck (looks something like https://www.holidaycheck.de/hi/haus-an-den-duenen/eeb089dc-cdba-3d20-ab90-c4db288e5dd7)
7. The plugin will open this URL and get your HolidayCheck-ID
8. Select your options
9. Save - and you're done!
